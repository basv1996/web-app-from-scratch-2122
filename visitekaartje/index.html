About Me

Front-ender in hart en nieren. Ik heb mijzelf de laatste jaren ontwikkeld als front-end developer. Hierbij let ik vooral op de usability. Dit houdt in dat het web voor iedereen toegankelijk moet zijn.

HTML

HTMl staat voor HyperText Markup Language. Dit is de "body" van elke website. Hierin worden alle teksten etc weergegeven. Het is de taal die gebruikt wordt om webpagina's op het internet te ontwerpen en te beschrijven.

CSS

CSS staat voor Cascading Style Sheet. Het is een hulpmiddel voor website ontwerpers om de vormgeving van een website vast te leggen. Deze taal beheers ik aardig goed en kan mijn weg er goed in vinden.

JavaScript

Met de programeertaal javaScript kunnen webpagina's interactief worden gemaakt. Deze taal beheers ik een beetje maar kan mijn weg er goed in vinden.
